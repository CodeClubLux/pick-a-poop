# directory structure

## data

This will contain static datasets

## bin

Scripts and other CLI goodies

## web

Main web directory for html files

### web/static

Static files

#### web/static/js

Main JavaScript directory. Can include user scripts and other JS frameworks

#### web/static/css

Main CSS directory

#### web/static/img

Main images directory

#### web/static/tmp

Temp directory
